::card-group
  :::card{icon="i-lucide-plug" title="API" to="/docs/composables/use-api"}
  CRUD API 工厂函数和数据请求。包含 3 个核心 composable。
  :::

  :::card{icon="i-lucide-toggle-left" title="Boolean State" to="/docs/composables/use-boolean"}
  语义化布尔状态管理（loading、submitting 等）。包含 11 个变体。
  :::

  :::card{icon="i-lucide-table-2" title="Table" to="/docs/composables/use-table"}
  表格数据管理、查询、分页、列管理、行选择等。包含 10 个 composable。
  :::

  :::card{icon="i-lucide-file-text" title="Form" to="/docs/composables/use-form"}
  表单状态管理和 Zod schema 自动生成。包含 2 个 composable。
  :::

  :::card{icon="i-lucide-wrench" title="Utilities" to="/docs/composables/utilities"}
  应用状态、主题、日期工具、EChart 集成、权限控制。包含 5 个 composable。
  :::
::

::note
所有 composable 均通过 Nuxt 模块自动注册到全局，你可以在任何 Vue 组件中直接使用，无需手动 `import`。
::

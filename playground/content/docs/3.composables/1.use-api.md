---
title: useApi
description: 标准化的 CRUD API 工厂函数，根据 basePath 自动生成完整的增删改查方法。
category: composable
---

## Request Methods

| Composable | Description |
| --- | --- |
| `useApiFetch<T>(url, options?)` | 底层 fetch 包装器，自动使用 `$api` 插件提供的 baseURL |
| `usePostFetch<T>(url, data?, customOptions?)` | POST 请求快捷方法 |
| `usePutFetch<T>(url, data?, customOptions?)` | PUT 请求快捷方法 |
| `useGetFetch<T>(url, customOptions?)` | GET 请求快捷方法 |
| `useDeleteFetch<T>(url, customOptions?)` | DELETE 请求快捷方法 |

## Factory Functions

`useApi<M>(basePath)` 组合所有工厂函数，返回完整的 `ApiGroup<M>`：

| Function | Description |
| --- | --- |
| `useApiCreate<M>` | 创建记录 |
| `useApiBatchCreate<M>` | 批量创建记录 |
| `useApiUpdate<M>` | 更新记录 |
| `useApiBatchUpdate<M>` | 批量更新记录 |
| `useApiGetById<M>` | 按 ID 获取记录 |
| `useApiGetByString<M>` | 按字符串参数获取记录 |
| `useApiDeleteById` | 按 ID 删除记录 |
| `useApiBatchDelete` | 批量删除记录 |
| `useApiCount<M>` | 查询计数 |

## Usage

```ts
// 1. 创建业务 API composable
export const useUserApi = () => useApi<Model.User>('/users')

// 2. 使用 API
const { create, update, getById, deleteById, list, count } = useUserApi()

// 3. 查询
const result = await list({
  whereQuery: {
    items: [
      { field: 'status', opr: 'eq', value: 'active' },
      { field: 'nickname', opr: 'like', value: '张' }
    ]
  },
  orderQuery: [
    { field: 'createdAt', order: 'desc' }
  ],
  pagination: { pageNum: 1, pageSize: 10 }
})
```

## useBizModel

Model 工具集，提供 ID 提取等快捷方法。

```ts
// useBizModel - 提取 ID 列表
const { extractIds } = useBizModel()
const ids = extractIds(selectedUsers) // number[]
```

::tip
`useApi` 内部依赖 `$api` 插件提供的 `baseURL`，请确保在 Nuxt 插件中正确配置了 API 基础地址。
::

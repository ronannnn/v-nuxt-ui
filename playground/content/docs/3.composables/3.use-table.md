## Composables

| Composable | Description |
| --- | --- |
| `useTable<T>(props)` | 表格核心编排器，组合所有子 composable，提供完整的表格状态管理 |
| `useProTableView<T>(props)` | 表格视图层，管理渲染和交互 |
| `useTableData<T>()` | 表格数据管理：fetch、refresh、cache |
| `useTableQuery<T>()` | 查询状态管理：whereQuery + orderQuery 的组合 |
| `useTablePagination()` | 分页状态管理：pageNum、pageSize、total |
| `useTableColumns<T>()` | 列管理：显示/隐藏、排序、固定（pin） |
| `useTableRowSelection<T>()` | 行选择管理：单选/全选/批量操作 |
| `useTableRowActions<T>()` | 行操作管理：编辑/复制/删除/自定义操作 |
| `useTableOpr<T>()` | 表格操作管理：CRUD 操作的统一处理 |
| `useTableWhereQuery<T>()` | Where 查询筛选管理 |

## Query Operators

`useTableQuery` 支持的 where 查询运算符：

| Operator | Description |
| --- | --- |
| `eq` | 等于 |
| `ne` | 不等于 |
| `like` | 模糊匹配 |
| `start_like` | 前缀匹配 |
| `in` | 包含在列表中 |
| `not_in` | 不包含在列表中 |
| `gt` | 大于 |
| `gte` | 大于等于 |
| `lt` | 小于 |
| `lte` | 小于等于 |
| `is_null` | 为空 |
| `is_not_null` | 不为空 |

## Usage

```vue
<ProTable
  name="users"
  cn-name="用户信息"
  :use-api-group="useUserApi"
  :biz-columns="columns"
  :on-edit-row-from-modal="handleEdit"
  expandable
  :export-excel="{ filename: 'users-export' }"
/>
```

::tip
通常你不需要直接使用这些 composable，`ProTable` 组件内部已经自动编排了所有子 composable。只有在需要完全自定义表格行为时才需要直接调用。
::

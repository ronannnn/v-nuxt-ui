---
title: useForm
description: 表单状态管理，配合 ProForm 组件使用。支持 Zod schema 自动生成、值 diff 对比和提交逻辑封装。
category: composable
---

## Composables

| Composable | Returns | Description |
| --- | --- | --- |
| `useFormValues<T>(raw, defaultValues?)` | `{ oldValues, newValues }` | 表单值管理，维护原始值和新值的双向绑定 |
| `useFormSubmission<T>(oldValues, newValues, close, save, apiGroup, ...)` | `{ onSubmit }` | 表单提交逻辑，自动判断新建/编辑，处理 diff 和 API 调用 |

## Usage

```ts
// 表单值管理
const { oldValues, newValues } = useFormValues(
  toRef(props, 'modelValue'),
  { status: 'active' } // 新建时的默认值
)

// 表单提交
const { onSubmit } = useFormSubmission(
  oldValues,
  newValues,
  (ok) => emit('close', ok),
  undefined, // save 函数（可选）
  apiGroup    // ApiGroup 实例
)
```

## With ProForm Template

```vue
<ProFormCreateModalTemplateWithApi
  :title="title"
  :fields="fields"
  :model-value="model"
  :api-group="useUserApi"
  :on-close="(ok) => emit('close', ok)"
/>
```

::tip
`ProFormCreateModalTemplateWithApi` 内部已经封装了 `useFormValues` 和 `useFormSubmission`，你只需传入 `fields`、`model-value` 和 `api-group` 即可。
::

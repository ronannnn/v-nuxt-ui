---
title: useBoolean
description: 语义化的布尔状态管理，所有变体基于 `useBoolean` 封装，提供更具表达力的 API。
category: composable
---

## All Variants

| Composable | Returns | Description |
| --- | --- | --- |
| `useBoolean(initValue?)` | `{ bool, setBool, setTrue, setFalse, toggle }` | 通用布尔状态管理 |
| `useLoading(initValue?)` | `{ loading, startLoading, endLoading }` | 加载状态 |
| `useFetching(initValue?)` | `{ fetching, startFetching, endFetching }` | 数据请求状态 |
| `useSubmitting(initValue?)` | `{ submitting, startSubmitting, endSubmitting }` | 表单提交状态 |
| `useDeleting(initValue?)` | `{ deleting, startDeleting, endDeleting }` | 删除操作状态 |
| `useUploading(initValue?)` | `{ uploading, startUploading, endUploading }` | 上传状态 |
| `useDownloading(initValue?)` | `{ downloading, startDownloading, endDownloading }` | 下载状态 |
| `useSearching(initValue?)` | `{ searching, startSearching, endSearching }` | 搜索状态 |
| `useExporting(initValue?)` | `{ exporting, startExporting, endExporting }` | 导出状态 |
| `useCopying(initValue?)` | `{ copying, startCopying, endCopying }` | 复制状态 |
| `useLoadingFn(fn)` | `{ loading, startLoading, endLoading, handleLoadingFn }` | 自动管理 loading 状态的异步函数包装 |

## Usage

```ts
// 基础用法
const { loading, startLoading, endLoading } = useLoading()

async function fetchData() {
  startLoading()
  try {
    const data = await api.list(query)
    // ...
  } finally {
    endLoading()
  }
}

// 模板中使用
// <UButton :loading="loading" />
```

## useLoadingFn

自动管理 loading 状态的异步函数包装，免去手动 `startLoading` / `endLoading`。

```ts
const { loading, handleLoadingFn } = useLoadingFn(async () => {
  await api.create(model)
})

// 调用时自动开始/结束 loading
await handleLoadingFn()
```

::tip
推荐使用 `useLoadingFn` 包装异步操作，避免手动管理 loading 状态时忘记在 `finally` 中调用 `endLoading`。
::

## useApp

应用级状态管理：响应式断点（isMobile、isSmallerThanMd）、locale、keepalive、主题设置面板等。

**Returns:** `{ appConfig, isMobile, updateLocale, resetAppConfig, ... }`

## useTheme

主题管理：颜色模式、Primary/Neutral 颜色、圆角、国际化语言列表。

**Returns:** `{ primary, neutral, radius, mode, locale, primaryColors, ... }`

```ts
const { primary, neutral, mode, primaryColors } = useTheme()

// 当前主色
console.log(primary.value) // 'blue'

// 切换暗色模式
const { updatePrimaryColor } = useApp()
updatePrimaryColor('green')
```

## useDate

日期工具集：dayjs <-> DateValue 转换、格式化、时区处理。

**Returns:** `{ dayjsToDateValue, dateValueToDayjs, formatTimeUnit, ... }`

```ts
const { dayjsToDateValue, dateValueToDayjs, formatTimeUnit } = useDate()

// dayjs → DateValue（用于 DatePicker）
const dateValue = dayjsToDateValue(dayjs())

// DateValue → dayjs（用于 API 提交）
const dayjsDate = dateValueToDayjs(dateValue)

// ISO 字符串 → DateValue
const dv = isoUtcStringToDateValue('2024-01-15T00:00:00Z')
```

## useEChart

ECharts 图表集成：通用配置项、系列合并、CSS 变量颜色解析、图表导出。

**Returns:** `{ mergeOption, mergeSeries, getCommonGridOption, exportChartAsImage, ... }`

::note
`useEChart` 需要安装可选依赖 `echarts`，并配合 `ProEChart` 组件使用。
::

## usePermission

权限控制：provide/inject 模式，配合 `ProPermissionWrapper` 组件使用。

**Returns:** `{ hasPermission, checkPermission }`

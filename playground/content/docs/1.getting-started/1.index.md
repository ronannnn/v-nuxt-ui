## 概览

V Nuxt UI 是一个基于 `@nuxt/ui` 构建的企业级 Nuxt 模块，提供开箱即用的业务组件和 composables，帮助你快速构建数据密集型后台管理系统。

::card-group
  :::card{icon="i-lucide-box" title="68+ 组件" to="/docs/components"}
  ProTable、ProForm、ProDeleteModal、ProDatePicker、ProScrollArea、ProDnd、ProEChart 等。
  :::

  :::card{icon="i-lucide-puzzle" title="44+ Composables" to="/docs/composables"}
  useApi、useBoolean、useTable、useForm、useDate、useTheme、usePermission 等。
  :::

  :::card{icon="i-lucide-text-cursor-input" title="18+ 表单字段类型"}
  input、select、async-select、date-picker、switch、dynamic-input、sql-editor 等。
  :::

  :::card{icon="i-lucide-filter" title="12+ 筛选运算符"}
  eq、ne、like、in、not_in、gt、gte、lt、lte、is_null、is_not_null、range 等。
  :::
::

## 核心组件

| 组件 | 说明 |
| --- | --- |
| [`ProTable`](/docs/components/table) | 企业级数据表格，支持 CRUD、分页、列管理、查询筛选、行选择、Excel 导出 |
| [`ProForm`](/docs/components/form) | 声明式表单构建器，18+ 字段类型、Zod 验证、模态框/页面模板 |
| [`ProDeleteModal`](/docs/components/delete-modal) | 通用删除确认模态框，支持批量删除 |
| [`ProDatePicker`](/docs/components/date-picker) | 增强日期选择器，支持快捷预设和范围选择 |
| [`ProScrollArea`](/docs/components/scroll-area) | 自定义滚动区域，支持边缘渐变、滚动检测和横向滚动提示 |
| [`ProDnd`](/docs/components/dnd) | 拖拽排序列表组件，基于 vue-draggable-plus |

## 快速开始

::note
请确保你已有一个 [Nuxt 4](https://nuxt.com) 项目，并安装了 `@nuxt/ui` v4。
::

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['v-nuxt-ui'],
  vNuxtUI: {
    prefix: 'Pro'
  }
})
```

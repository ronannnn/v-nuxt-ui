## 安装依赖

安装 `v-nuxt-ui` 和必需的 peer dependencies。

::code-group
```bash [pnpm]
pnpm add v-nuxt-ui @nuxt/ui @nuxtjs/i18n @vueuse/nuxt
```

```bash [npm]
npm install v-nuxt-ui @nuxt/ui @nuxtjs/i18n @vueuse/nuxt
```

```bash [yarn]
yarn add v-nuxt-ui @nuxt/ui @nuxtjs/i18n @vueuse/nuxt
```
::

### 可选依赖

根据你使用的功能，安装对应的可选依赖：

```bash
pnpm add zod                  # 表单验证
pnpm add echarts               # 图表组件
pnpm add exceljs               # Excel 导出
pnpm add vue-draggable-plus    # 拖拽组件
```

## 配置模块

在 `nuxt.config.ts` 中添加模块：

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['v-nuxt-ui'],

  vNuxtUI: {
    prefix: 'Pro' // 组件前缀，默认 'V'
  }
})
```

## 配置 CSS

在主 CSS 文件中导入 Tailwind CSS、`@nuxt/ui` 和 `v-nuxt-ui`：

```css
/* app/assets/css/main.css */
@import "tailwindcss";
@import "@nuxt/ui";
@import "v-nuxt-ui";
```

## 模块选项

| 选项 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `prefix` | `string` | `'V'` | 组件名称前缀。例如设为 `'Pro'` 后，组件将注册为 `ProTable`、`ProForm` 等。 |

## Peer Dependencies

### 必需

| 包名 | 版本 |
| --- | --- |
| `@nuxt/ui` | >= 4.5.0 |
| `@nuxtjs/i18n` | >= 10.2.3 |
| `@vueuse/nuxt` | >= 14.2.1 |
| `nuxt` | >= 4.3.1 |

### 可选

| 包名 | 用途 |
| --- | --- |
| `zod` | 表单验证 |
| `echarts` | EChart 图表 |
| `exceljs` | Excel 导出 |
| `vue-draggable-plus` | 拖拽排序 |
| `cronstrue` | Cron 表达式 |

::note
所有组件和 composables 均通过 Nuxt 模块自动注册，无需手动 `import`。
::

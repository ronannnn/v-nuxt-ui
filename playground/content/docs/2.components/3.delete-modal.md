### Single Delete

```ts
const overlay = useOverlay()
const deleteModal = overlay.create(ProDeleteModal)

async function handleDelete(row) {
  const ok = await deleteModal.open({
    ids: [row.id],
    onDelete: async (ids) => {
      return await useUserApi().batchDelete(ids)
    }
  }).result
  if (ok) {
    // 刷新数据
  }
}
```

::tip
ProDeleteModal 通常由 `ProTable` 自动管理，你不需要手动创建。只有在自定义删除流程时才需要直接使用。
::

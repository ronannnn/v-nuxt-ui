## Usage

### ProBadgeCodeModal

点击徽章打开模态框，展示语法高亮的代码内容。支持 JSON、SQL、纯文本，内置复制按钮。

::component-code
---
slug: badge-code-modal
props:
  data: '{"name": "Alice", "age": 30}'
  lang: json
  title: API Response
items:
  lang:
    - json
    - sql
    - text
---
::

### ProBadgeCodeTooltip

悬浮徽章预览代码内容，使用 Shiki 语法高亮。

::component-code
---
slug: badge-code-tooltip
props:
  data: '{"key": "value"}'
  lang: json
items:
  lang:
    - json
    - sql
    - text
---
::

::note
两个组件都使用 Shiki 进行语法高亮，支持 `github-light` / `github-dark` 主题自动切换。`ProBadgeCodeModal` 额外提供复制到剪贴板功能。
::

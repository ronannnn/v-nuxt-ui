---
title: ProSpin
description: 加载状态覆盖层组件。用于在异步操作期间覆盖内容区域并显示旋转加载动画。
category: element
---

## Usage

### Loading

::component-code
---
props:
  loading: true
slots:
  default: |

    <div class="p-8 text-center">Content that can be loading...</div>
items:
  loading:
    - true
    - false
---

#default
:::div{class="p-8 text-center"}
Content that can be loading...
:::
::

::note
覆盖层使用 `fade` 过渡动画，带有半透明背景和居中的旋转图标。
::

## Usage

### ProButtonDropdown

基于 `UPopover` + `UCommandPalette` 的下拉选择按钮，支持单选/多选、搜索和批量操作。

::component-code
---
slug: button-dropdown
props:
  groups:
    - id: fruits
      items:
        - id: apple
          label: Apple
        - id: banana
          label: Banana
        - id: cherry
          label: Cherry
  multiple: false
slots:
  default: |
    <UButton>Select...</UButton>
items:
  multiple:
    - true
    - false
---
::

### ProButtonTheme

主题选择按钮，可展示颜色点、语言标志或自定义前缀内容。

::component-code
---
slug: button-theme
props:
  label: Blue Theme
  chip: blue
  selected: false
items:
  selected:
    - true
    - false
---
::

## Usage

### ProButtonDropdown

基于 `UPopover` + `UCommandPalette` 的下拉选择按钮，支持单选/多选、搜索和批量操作。

::component-code
---
slug: button-dropdown
props:
  groups:
    - id: fruits
      items:
        - value: apple
          label: Apple
        - value: banana
          label: Banana
        - value: cherry
          label: Cherry
  multiple: false
slots:
  default: |

    <UButton>Select</UButton>
items:
  multiple:
    - true
    - false
---

#default
:u-button{label="Select"}
::

### ProButtonTheme

主题选择按钮，可展示颜色点、语言标志或自定义前缀内容。

::component-code
---
slug: button-theme
props:
  label: Blue Theme
  chip: blue
  selected: false
items:
  selected:
    - true
    - false
---
::

### ProButtonConfirm

二次确认按钮，第一次点击进入确认态，第二次点击触发 `confirm` 事件。按钮宽度会在两个状态之间过渡。

::component-code
---
slug: button-confirm
props:
  button:
    icon: i-lucide-x
    color: neutral
    variant: outline
  confirmButton:
    label: 确认
    color: error
    variant: subtle
---
::

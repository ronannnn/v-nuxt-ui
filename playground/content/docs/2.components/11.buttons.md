## ProButtonChip

带颜色指示点的小型标签按钮，支持禁用状态。

### Example

```vue
<ProButtonChip label="Active" color="#22c55e" :on-click="handleClick" />
<ProButtonChip label="Disabled" color="#ef4444" disabled :on-click="() => {}" />
```

## ProButtonDropdown

基于 `UPopover` + `UCommandPalette` 的下拉选择按钮，支持单选/多选、搜索和批量操作。

### Example

```vue
<ProButtonDropdown v-model="selected" :groups="groups">
  <UButton>Select...</UButton>
</ProButtonDropdown>
```

### Props

:component-props{slug="button-dropdown"}

### Slots

:component-slots{slug="button-dropdown"}

## ProButtonTheme

主题选择按钮，可展示颜色点、语言标志或自定义前缀内容。

### Example

```vue
<ProButtonTheme label="Blue" chip="blue" :selected="true" />
<ProButtonTheme label="Settings" icon="i-lucide-settings" />
```

:component-props{slug="button-theme"}

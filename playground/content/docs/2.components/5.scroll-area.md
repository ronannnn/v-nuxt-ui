---
title: ProScrollArea
description: 自定义滚动区域组件，基于 reka-ui ScrollArea，支持边缘渐变效果、滚动检测、横向滚动提示和编程式滚动。
category: layout
---

## Usage

### Edge fade

::component-code
---
slug: scroll-area
props:
  enableTopTransparent: true
  enableBottomTransparent: true
slots:
  default: |
    <div class="h-48">
      <div v-for="i in 20" :key="i" class="p-3 border-b border-muted">Item {{ i }}</div>
    </div>
items:
  enableTopTransparent:
    - true
    - false
  enableBottomTransparent:
    - true
    - false
---
::

### Horizontal

::component-code
---
slug: scroll-area
props:
  enableLeftTransparent: true
  enableRightTransparent: true
  enableHorizontalTooltip: true
slots:
  default: |
    <div class="flex gap-4 w-max p-4">
      <div v-for="i in 15" :key="i" class="w-32 h-16 shrink-0 rounded bg-muted flex items-center justify-center">Col {{ i }}</div>
    </div>
items:
  enableHorizontalTooltip:
    - true
    - false
---
::

### Exposed Methods

| Method | Description |
| --- | --- |
| `scrollHorizontally(targetX)` | 水平滚动到绝对位置 |
| `scrollVertically(targetY)` | 垂直滚动到绝对位置 |
| `scrollHorizontallyBy(deltaX, behavior?)` | 水平增量滚动 |
| `scrollVerticallyBy(deltaY, behavior?)` | 垂直增量滚动 |

::tip
边缘渐变效果会自动根据滚动位置显示/隐藏，当内容滚动到顶部时，顶部渐变消失；滚动到底部时，底部渐变消失。
::

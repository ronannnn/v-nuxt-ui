## Examples

### Vertical Scrolling

```vue
<ProScrollArea
  enable-top-transparent
  enable-bottom-transparent
  class="h-64"
>
  <div v-for="item in items" :key="item.id" class="p-3 border-b">
    {{ item.label }}
  </div>
</ProScrollArea>
```

### Horizontal Scrolling

```vue
<ProScrollArea
  enable-left-transparent
  enable-right-transparent
  enable-horizontal-tooltip
  class="h-20"
>
  <div class="flex gap-4 w-max">
    <div v-for="col in columns" :key="col" class="w-40 shrink-0">
      {{ col }}
    </div>
  </div>
</ProScrollArea>
```

### Exposed Methods

| Method | Description |
| --- | --- |
| `scrollHorizontally(targetX)` | 水平滚动到绝对位置 |
| `scrollVertically(targetY)` | 垂直滚动到绝对位置 |
| `scrollHorizontallyBy(deltaX, behavior?)` | 水平增量滚动 |
| `scrollVerticallyBy(deltaY, behavior?)` | 垂直增量滚动 |

::tip
边缘渐变效果会自动根据滚动位置显示/隐藏，当内容滚动到顶部时，顶部渐变消失；滚动到底部时，底部渐变消失。
::

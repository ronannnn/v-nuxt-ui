---
title: ProEmpty
description: 空状态占位组件，当列表或容器没有数据时展示。基于 Nuxt UI 的 UEmpty 封装，提供中文默认值。
category: element
---

## Usage

### Default

::component-code
---
props: {}
---
::

### Icon

::component-code
---
props:
  icon: i-lucide-search
  title: No results found
---
::

### Variant

::component-code
---
props:
  variant: outline
  icon: i-lucide-inbox
  title: 暂无数据
items:
  variant:
    - naked
    - outline
    - solid
    - soft
    - subtle
    - ghost
---
::

### Size

::component-code
---
props:
  size: md
  title: 暂无数据
items:
  size:
    - xs
    - sm
    - md
    - lg
    - xl
---
::

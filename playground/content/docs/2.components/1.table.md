::

## Features

- **声明式列定义** — `VColumn<T>` 扩展 NuxtUI TableColumn，支持 `filterOption`、`sortOption`、`initHide`、`exportCell`
- **查询筛选** — 支持 input、input-number、select、async-select、date-picker 等筛选类型，12+ 运算符
- **多列排序** — orderQuery 支持多列排序，可设置默认排序方向
- **CRUD 操作** — 内置创建、编辑、复制、删除操作，通过 `useApiGroup` 绑定 API
- **分页** — 内置分页组件，自动管理 pageNum、pageSize
- **行选择与批量删除** — 支持行复选框、全选、批量删除确认
- **列管理** — 拖拽排序列、显示/隐藏列、列固定（pin）
- **行展开** — expandable + expandVNode 自定义展开内容
- **自定义行操作** — extraRowActions 添加额外操作按钮
- **Excel 导出** — 内置 Excel 导出功能，支持自定义 filename、exportCell
- **ProTablePage** — 包含 UDashboardPanel 的完整页面模式

## Examples

### API Group

```ts
// composables/api/useUserApi.ts
export const useUserApi = () => useApi<Model.User>('/users')

// useApi 返回的 ApiGroup<T> 包含：
// - create(data: T): Promise<RequestResult<T>>
// - update(data: T): Promise<RequestResult<T>>
// - getById(id: number): Promise<RequestResult<T>>
// - deleteById(id: number): Promise<RequestResult<void>>
// - batchDelete(ids: number[]): Promise<RequestResult<void>>
// - count(query): Promise<RequestResult<number>>
// - list(query): Promise<RequestResult<PageResult<T>>>
// - countAndList(query): Promise<...>
```

### Column Definition

```ts
const columns: VColumn<Model.User>[] = [
  {
    accessorKey: 'nickname',
    header: '姓名',
    sortOption: true,
    filterOption: {
      type: 'input',
      initHide: false
    }
  },
  {
    accessorKey: 'departmentId',
    header: '部门',
    filterOption: {
      type: 'async-select',
      listApi: useDepartmentApi().list,
      searchFields: ['name'],
      labelField: 'name',
      multiple: true,
      defaultOpr: 'in'
    }
  },
  {
    accessorKey: 'status',
    header: '状态',
    filterOption: {
      type: 'select',
      items: statusOptions
    }
  },
  {
    accessorKey: 'entryDate',
    header: '入职时间',
    sortOption: true,
    filterOption: { type: 'date-picker' }
  }
]
```

### Basic usage

```vue
<ProTable
  name="playground-users"
  cn-name="用户信息"
  :use-api-group="useUserApi"
  :biz-columns="columns"
  :on-edit-row-from-modal="openEditModal"
  expandable
  :expand-v-node="expandVNode"
  :export-excel="{
    filename: 'users-export',
    filenameWithDateTime: true
  }"
/>
```

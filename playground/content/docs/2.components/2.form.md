## Field Types

::note
所有字段类型均通过 `VFormFieldProps` 的 `type` 属性声明，ProForm 会自动渲染对应的 UI 组件。
::

| Type | Description |
| --- | --- |
| `input` | 文本输入框 |
| `input-number` | 数字输入框 |
| `input-pwd` | 密码输入框 |
| `input-string-number` | 字符串数字输入框 |
| `textarea` | 多行文本 |
| `select` | 下拉选择 |
| `async-select` | 异步远程搜索选择 |
| `async-object-select` | 异步对象选择 |
| `async-tree-select` | 异步树形选择 |
| `tree-select-transfer` | 树形穿梭框 |
| `switch` | 开关 |
| `button-switch` | 按钮式开关 |
| `date-picker` | 日期选择器 |
| `dynamic-input` | 动态输入列表 |
| `dynamic-object-input` | 动态对象输入列表 |
| `sql-editor` | SQL 代码编辑器 |
| `separator` | 分隔线 |
| `custom` | 自定义渲染 |



### Modal Form Templates

::code-group
```vue [WithApi (推荐)]
<!-- 使用 ProFormCreateModalTemplateWithApi 自动绑定 API -->
<ProFormCreateModalTemplateWithApi
  :title="isEdit ? '编辑用户' : '新建用户'"
  :fields="fields"
  :model-value="model"
  :api-group="useUserApi"
  :on-close="(ok) => emit('close', ok)"
/>
```

```vue [Manual]
<!-- 或使用 ProFormCreateModalTemplate 手动控制提交 -->
<ProFormCreateModalTemplate
  title="用户信息"
  :fields="fields"
  :model-value="model"
  :on-submit="handleSubmit"
  :on-close="handleClose"
/>
```
::

::tip
`ProFormCreateModalTemplateWithApi` 内部封装了 `useFormValues` 和 `useFormSubmission`，自动处理新建/编辑判断和 diff 提交。
::

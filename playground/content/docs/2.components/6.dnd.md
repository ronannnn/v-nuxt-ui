---
title: ProDnd
description: 拖拽排序列表组件，基于 vue-draggable-plus。支持拖拽手柄、分组跨列表拖拽、克隆、移动端触摸延迟等。
category: element
---

## Usage

### Default

::component-code
---
slug: dnd
props:
  modelValue:
    - id: 1
      label: 首页
    - id: 2
      label: 仪表盘
    - id: 3
      label: 设置
  handle: ".drag-handle"
slots:
  default: |
    <div v-for="item in items" :key="item.id" class="flex items-center gap-2 p-3 border-b border-muted">
      <UIcon name="i-lucide-grip-vertical" class="drag-handle cursor-grab text-muted" />
      <span>{{ item.label }}</span>
    </div>
---
::

::note
ProDnd 是 **Client Only** 组件，需要在 `<ClientOnly>` 中使用或确保 `ssr: false`。
::

### Cross-list Dragging

```vue
<div class="grid grid-cols-2 gap-4">
  <ProDnd v-model="listA" group="shared">
    <div v-for="item in listA" :key="item.id">{{ item.label }}</div>
  </ProDnd>
  <ProDnd v-model="listB" group="shared">
    <div v-for="item in listB" :key="item.id">{{ item.label }}</div>
  </ProDnd>
</div>
```

::tip
使用 `group` 属性设置相同的分组名，即可实现跨列表拖拽。设置 `group` 为对象 `{ name: 'shared', pull: 'clone', put: false }` 可实现克隆拖拽。
::

---
title: ProPermissionWrapper
description: 权限控制包装器组件，通过注入的权限检查函数来控制内容的可见性或可交互性。支持 hide（隐藏）和 disable（禁用点击）两种模式。
category: layout
---

## Usage

### Mode

::component-code
---
slug: permission-wrapper
props:
  permission: admin:write
  mode: disable
slots:
  default: |
    <UButton>Create</UButton>
items:
  mode:
    - disable
    - hide
---
::

### Fallback

::component-code
---
slug: permission-wrapper
props:
  permission: admin:delete
  mode: hide
  fallback: true
slots:
  default: |
    <UButton color="error">Delete</UButton>
items:
  fallback:
    - true
    - false
---
::

## Examples

### Setup permission check

```vue
<script setup>
// 应用层提供权限检查函数
provide('vui:hasPermissions', (perms) => {
  return userPermissions.includes(perms)
})
</script>

<!-- 禁用模式：无权限时点击被拦截 -->
<ProPermissionWrapper permission="admin:write" mode="disable">
  <UButton>Create</UButton>
</ProPermissionWrapper>

<!-- 隐藏模式：无权限时显示 fallback -->
<ProPermissionWrapper permission="admin:delete" mode="hide" fallback>
  <UButton color="error">Delete</UButton>
  <template #fallback>
    <UBadge color="warning">No permission</UBadge>
  </template>
</ProPermissionWrapper>
```

::tip
宿主应用需要通过 `provide('vui:hasPermissions', fn)` 注入权限检查函数。如果未提供，默认允许所有权限。
::

---

## Usage

### Default

::component-code
---
slug: date-picker
props:
  placeholder: 选择日期
---
::

### Range

::component-code
---
slug: date-picker
props:
  range: true
  placeholder: 选择日期范围
items:
  range:
    - true
    - false
---
::

## Features

- 支持单日期和日期范围选择
- 弹出式日历面板（基于 reka-ui UCalendar）
- 支持手动输入日期字符串
- 内置"今天"和"清空"快捷按钮
- 可自定义快捷日期预设（shortcuts）
- 响应式布局（移动端单月，桌面端双月）
- 支持并列按钮（peerButtons）

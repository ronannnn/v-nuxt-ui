## Features

- 支持单日期和日期范围选择
- 弹出式日历面板（基于 reka-ui UCalendar）
- 支持手动输入日期字符串
- 内置"今天"和"清空"快捷按钮
- 可自定义快捷日期预设（shortcuts）
- 响应式布局（移动端单月，桌面端双月）
- 支持并列按钮（peerButtons）

## Examples

### Single Date

```vue
<script setup>
import type { DateValue } from 'reka-ui'

const date = ref<DateValue | null>(null)
</script>

<template>
  <ProDatePicker v-model="date" placeholder="选择日期" />
</template>
```

### Range Selection

```vue
<script setup>
import type { DateRange } from 'reka-ui'

const range = ref<DateRange>({
  start: undefined,
  end: undefined
})
</script>

<template>
  <ProDatePicker
    v-model="range"
    range
    placeholder="选择日期范围"
  />
</template>
```

### Shortcuts

```vue
<ProDatePicker
  v-model="date"
  :shortcuts="[
    {
      label: '最近7天',
      dateFn: () => getRecentDateRange(7)
    },
    {
      label: '最近30天',
      dateFn: () => getRecentDateRange(30)
    }
  ]"
/>
```

::note
`getRecentDateRange` 和 `getCurrentMonthRange` 是 `useDate()` composable 提供的快捷方法。
::

---
title: ProSimpleTable
description: 轻量级只读数据表格，基于 UTable 封装。提供固定列阴影效果、自动 Badge 渲染，适合静态数据展示。
category: data
---

## Usage

### Default

::component-code
---
slug: simple-table
props:
  data:
    - id: 1
      name: Alice
      status: active
    - id: 2
      name: Bob
      status: inactive
  bizColumns:
    - accessorKey: id
      header: ID
    - accessorKey: name
      header: Name
    - accessorKey: status
      header: Status
items:
  singleRow:
    - true
    - false
---
::

---
title: ProCircleProgress
description: SVG 圆形进度指示器，支持多色弧形段、自定义笔画宽度和中心文字。
category: element
---

## Usage

### Default

::component-code
---
slug: circle-progress
props:
  arcs:
    - percent: 75
      color: primary
  text: "75%"
  textColor: primary
---
::

### Multi-segment

::component-code
---
slug: circle-progress
props:
  arcs:
    - percent: 40
      color: primary
    - percent: 30
      color: success
    - percent: 20
      color: warning
    - percent: 10
      color: error
  text: Stats
---
::

### Stroke Width

::component-code
---
slug: circle-progress
props:
  arcs:
    - percent: 60
      color: info
  text: "60%"
  strokeWidth: 4
---
::

## Examples

### Dashboard usage

```vue
<ProCircleProgress
  :arcs="[
    { percent: 40, color: 'primary' },
    { percent: 30, color: 'success' },
    { percent: 20, color: 'warning' },
    { percent: 10, color: 'error' }
  ]"
  text="Stats"
/>
```

::note
当所有 `percent` 值 ≤ 1 时，会自动乘以 100 作为百分比处理。所有弧段百分比会归一化使总和为 100（完整圆）。
::
